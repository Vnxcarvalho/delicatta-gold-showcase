
# Landing Page - Delicatta Joalheria

## Visão Geral
Catálogo elegante e minimalista para a joalheria Delicatta, com tema verde escuro e dourado, 84 espaços para produtos e sistema de filtros combinados.

---

## 1. Header & Navegação
- Logo da Delicatta centralizada no topo
- Menu simples com links para seções da página (Produtos, Sobre, Contato)
- Design clean com fundo verde escuro e detalhes dourados

## 2. Seção Hero (Banner Principal)
- Banner elegante de boas-vindas
- Destaque para a marca com uma frase de impacto
- Botão para rolar até a seção de produtos

## 3. Sistema de Filtros
**Filtros por Material (podem ser combinados):**
- Ouro
- Prata  
- Aço

**Filtros por Tipo de Joia (podem ser combinados):**
- Brincos
- Alianças
- Pulseiras
- Colares

- Botão para limpar todos os filtros
- Contador de produtos exibidos

## 4. Catálogo de Produtos
- **84 espaços para produtos** organizados em grade responsiva
- Cada card de produto exibe:
  - Espaço para foto (placeholder fácil de substituir)
  - Nome do produto
  - Preço
  - Badge de disponibilidade (Disponível / Esgotado)
  - Tags de material e tipo
- **Paginação "Ver mais/Ver menos"**: exibe inicialmente 12 produtos, com opção de carregar mais
- Layout responsivo: 4 colunas desktop, 2 tablet, 1 mobile

## 5. Seção de Contato
- Informações de contato da loja
- Links para redes sociais (WhatsApp, Instagram)
- Mapa ou endereço (opcional)

## 6. Footer
- Logo pequena
- Copyright
- Links úteis

---

## Facilidade de Edição
- **Arquivo de dados centralizado**: todos os 84 produtos em um único arquivo com comentários explicando cada campo
- **Exemplo de produto comentado** mostrando como preencher cada campo
- **Cores do tema em variáveis CSS** para fácil personalização
- **Comentários no código** explicando cada seção e componente

## Estrutura de Dados do Produto (exemplo)
Cada produto terá campos simples para você editar:
- ID (número único)
- Nome
- Preço
- Imagem (caminho da foto)
- Material (ouro, prata, aço)
- Tipo (brinco, aliança, pulseira, colar)
- Disponível (sim/não)
